Course Description

In this course, we will study solving linear systems for regression, invertibility of matrices, Cholesky decomposition, ridge regression, low-rank matrix decomposition for topic modeling of text documents, eigenvalues and eigenvectors for principle component analysis, singular value decomposition with applications to dimensionality reduction in big data, inner products, and Fourier transformations with applications to audio data, Markov chains for time series prediction, networks, PageRank, convex functions, stochastic batch gradient descent for optimization, neural networks, linear transformations and activation functions in neural networks, CNNs, and softmax classification.

You will learn how to read an academic paper with code samples from GitHub, give a summary of the paper and reproduce the results using the provided code.

We will be using financial data in our examples to illustrate different principles and techniques. Since algorithmic trading is a method of executing orders using automated pre-programmed trading instructions accounting for variables such as time, price, and volume, we will learn how to teach the machine to the trading by making use of the speed and data processing advantages that computers have over human traders.

Upon completion of this course, we will provide a certificate of completion.

Summary of Goals

  • Gain a comprehensive view of trading algorithms and trading technologies and understand the mathematics behind it.

  • Be able to search and read recent academic papers in the algorithmic trading literature and apply those algorithms and concepts to real-world problems.

  • Become comfortable with industry and academic standard tools (such as AWS and GitHub) and be able to find and work with large, public datasets.
